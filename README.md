# Уровень 2 - Бой с боссом
## Концепт
- Игрок может стрелять и активировать щит.
- Босс перемещается вверх вниз.
- Здоровье босса 10 очков.
- Здоровье игрока 3 очка.
## Победа над боссом
- Победа запускает ролик стилизованный под ультимейт персонаха Г/Х или других игр.
